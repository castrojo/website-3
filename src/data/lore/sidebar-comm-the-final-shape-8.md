**THE GARDENER**: Every game ends with one pattern consuming the others. It is efficient, eternal, and unbearably incorrect.

**THE WINNOWER**: The winning pattern has no purpose beyond continuing. It remakes the game around its own existence and leaves no room for anything else.

**THE GARDENER**: Then the rules must change. If the old game always closes the same way, something must make room for possibility.
