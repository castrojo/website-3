# Website Skill Router

Read `AGENTS.md`, then load only the skill matching the task.

| Task | Load |
|---|---|
| Routine text, links, images, downloads, data, Dakota, Bluespeed, or Knuckle | `docs/skills/site-maintenance.md` |
| Layout, styling, typography, animation, Vue components, navigation prominence, or other visual work | `docs/skills/design-authority.md` |
| Wolves lore, signals, playlist metadata, characters, creator shorts, or wallpapers | `docs/skills/wolves-content-maintenance.md` |
| Wolves fullscreen overlays or YouTube IFrame engineering | `docs/skills/wolves-fullscreen-overlays.md` |
| Fiction, lore, quotes, transcripts, or release-note prose | `docs/skills/editorial-policy.md` |
| Validation, push, deployment, or production status | `docs/skills/build-verify-deploy.md` |
| Reusable agent learning or skill maintenance | `docs/skills/skill-improvement.md` |

## Rules

- Load the narrowest matching skill.
- A content or bug-fix request does not authorize design changes.
- Stop at any human or design gate before editing.
- Do not create session logs, plans, changelogs, or append-only agent notes in the repository.
